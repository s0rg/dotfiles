# dotfiles
My dotfiles

This repo is managed via [this](https://www.atlassian.com/git/tutorials/dotfiles) method

# software

- wm: `i3`
- shell: `bash`
- editor: `vim`
- music: `cmus`
- menus: `rofi`
- terminal: `kitty`
- notifications: `dunst`
- theme generation: `pywal`
- media auto-mount: `udiskie`
